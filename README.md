## Projeto POO - Cadastro de Cliente

## Descrição

Este projeto foi desenvolvido com o objetivo de praticar os conceitos de **Programação Orientada a Objetos**

A aplicação realiza um cadastro simples de cliente via terminal, solicitando informações básicas como **nome** e **e-mail**

---

## Funcionalidades

* Entrada de dados (`Scanner`)
* Validação de campos
* Uso de classes
* Estrutura básica de um sistema orientado a objetos

---

##  Como funciona

O programa executa os seguintes passos:

1. Solicita o nome do usuário
2. Valida se o campo não está vazio
3. Solicita o e-mail
4. Valida se o campo não está vazio
5. Armazena os dados no objeto Cliente`
6. Recebe dados do Pedido
7. Gera um Numero de pedido
8. Retorna com dados e valor total do pedido
